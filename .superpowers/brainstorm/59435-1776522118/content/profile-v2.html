Unit conversion behaviour
• Toggle lbs → kg: value converts instantly (185 lbs → 83.9 kg). Input field updates live.
• Toggle ft/in → m: 5' 11" → 1.80 m. Two ft/in fields collapse to one m field.
• If user types a value in the wrong unit before toggling, the entered number is re-interpreted and converted.
• Preference saved in SharedPreferences — the app remembers your chosen unit system across sessions.
